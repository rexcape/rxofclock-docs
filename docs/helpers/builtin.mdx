## if

可使用 `#if` 助手来根据条件渲染，如果其参数返回 `false`、`undefined`、`null`、`''`、`0` 或 `[]`，handlebars 将不会渲染该代码块

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
<div class="entry">
{{#if author}}
<h1>{{firstName}} {{lastName}}</h1>
{{/if}}
</div>
```

```js
{
  author: true,
  firstName: "Yehuda",
  lastName: "Katz",
}
```

</TabItem>
<TabItem value="output" label="输出">

```text
<div class="entry">
<h1>Yehuda Katz</h1>
</div>
```

</TabItem>
</Tabs>

可以通过标注 else 的方式指定要运行的模板

```handlebars
<div class="entry">
{{#if author}}
<h1>{{firstName}} {{lastName}}</h1>
{{else}}
<h1>Unknown Author</h1>
{{/if}}
</div>
```

### includeZero

设置 includeZero 可以将 0 视为非空值，这可以控制 0 在条件判断时作为 true 还是 false

```handlebars
{{#if 0 includeZero=true}}
<h1>这里会渲染</h1>
{{/if}}
```

### 子表达式

建议使用助手代码向模板添加逻辑。可以编写助手代码并将其应用到子表达式中

例如在检查变量的初始化时, 内置的 `#if` 检查可能不合适, 因为它对空集合返回 false

可以写一段助手代码来检查 undefined

```js
Handlebars.registerHelper('isdefined', function (value) {
  return value !== undefined;
});
```

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
{{#if (isdefined value1)}}true{{else}}false{{/if}}
{{#if (isdefined value2)}}true{{else}}false{{/if}}
```

```js
{ value1: {} }
```

</TabItem>
<TabItem value="output" label="输出">

```text
true
false
```

</TabItem>
</Tabs>



## unless

`unless` 助手与 if 相反，如果返回 false 将会渲染代码块

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
<div class="entry">
{{#unless license}}
<h3 class="warning">WARNING: This entry does not have a license!</h3>
{{/unless}}
</div>
```

```js
{}
```

</TabItem>
<TabItem value="output" label="输出">

```text
<div class="entry">
<h3 class="warning">WARNING: This entry does not have a license!</h3>
</div>
```

</TabItem>
</Tabs>


## each

`each` 助手代码用于遍历列表，在块内可以使用 this 来引用被迭代的元素

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
<ul class="people_list">
  {{#each people}}
    <li>{{this}}</li>
  {{/each}}
</ul>
```

```js
{
  people: [
    "Yehuda Katz",
    "Alan Johnson",
    "Charles Jolley",
  ],
}
```

</TabItem>
<TabItem value="output" label="输出">

```text
<ul class="people_list">
    <li>Yehuda Katz</li>
    <li>Alan Johnson</li>
    <li>Charles Jolley</li>
</ul>
```

</TabItem>
</Tabs>

可以在任意上下文中使用 this 指针引用当前上下文

可以选择提供一个 else, 该代码块将只会在列表为空时显示

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
{{#each paragraphs}}
<p>{{this}}</p>
{{else}}
<p class="empty">No content</p>
{{/each}}
```

```js
{}
```

</TabItem>
<TabItem value="output" label="输出">

```text
<p class="empty">No content</p>
```

</TabItem>
</Tabs>

遍历项目时, 可以通过 `{{@index}}` 引用当前循环的索引

```handlebars
{{#each array}} {{@index}}: {{this}} {{/each}}
```

对于对象迭代, 可以使用 `{{@key}}` 引用当前键名

```handlebars
{{#each object}} {{@key}}: {{this}} {{/each}}
```

在数组上进行迭代时，通过 `@first` 和 `@last` 记录迭代的第一项和最后一项

嵌套的每个块都可以通过基于深度的路径访问迭代变量。例如要访问父级索引可以使用 `{{@../inedx}}`


## with

`with` 助手代码用于更改块的上下文表达式

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
{{#with person}}
{{firstname}} {{lastname}}
{{/with}}
```

```js
{
  person: {
    firstname: "Yehuda",
    lastname: "Katz",
  },
}
```

</TabItem>
<TabItem value="output" label="输出">

```text
Yehuda Katz
```

</TabItem>
</Tabs>

with 还能与代码块参数一起引用以定义当前代码块中的已知引用。例如，上面的例子可以转换成

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
{{#with city as | city |}}
  {{#with city.location as | loc |}}
    {{city.name}}: {{loc.north}} {{loc.east}}
  {{/with}}
{{/with}}
```

```js
{
  city: {
    name: "San Francisco",
    summary: "San Francisco is the <b>cultural center</b> of <b>Northern California</b>",
    location: {
      north: "37.73,",
      east: -122.44,
    },
    population: 883305,
  },
}
```

</TabItem>
<TabItem value="output" label="输出">

```text
San Francisco: 37.73, -122.44
```

</TabItem>
</Tabs>



## lookup

`lookup` 助手代码允许使用 Handlebars 变量进行动态的参数解析，这对于解析数组索引的值非常有用

<Tabs>
<TabItem value="input" label="输入" default>

```handlebars
{{#each people}}
   {{.}} lives in {{lookup ../cities @index}}
{{/each}}
```

```js
{
  people: ["Nils", "Yehuda"],
  cities: [
    "Darmstadt",
    "San Francisco",
  ],
}
```

</TabItem>
<TabItem value="output" label="输出">

```text
   Nils lives in Darmstadt
   Yehuda lives in San Francisco
```

</TabItem>
</Tabs>
